# Portfolio Website for `bupedev`

This repository hosts a portfolio site for [bupedev](https://bupe.dev).

## Development

Install dependencies and start the dev server:

```bash
npm install
npm run dev
```

Build the static site:

```bash
npm run build
```

## Content

- Page content lives in [`src/pages`](src/pages)
- CV entries live in [`src/content/cv`](src/content/cv)
- Site metadata lives in [`src/data/site.ts`](src/data/site.ts)
